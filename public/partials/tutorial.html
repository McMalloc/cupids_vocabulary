Every OKCupid user is asked to add some general information to their profile. For the traits and features listed in this column the user can select from a finite pool of answers, like "yes" and "when drinking" for the feature smokes.

Additionally users can write in their own words about themselves. They are given certain categories—the essays listed above—to express a more personal side.

Selecting an essay will show the words which were most frequently used by the users to describe themselves.

Selecting features will filter the essays that are taken into account. For example, selecting "asian" for the feature "ethnicity" will only count the words of users who have selected asian as part of their ethnicity.

Tip: You can select multiple features by pressing Ctrl / Cmd while clicking when it makes sense (people can stem from multiple ethnicities but cannot be a smoker and a non-smoker at the same time.)
