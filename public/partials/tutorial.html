Every OKCupid user is asked to add some general information to their profile. For the traits and features listed in this column the user can select from a finite pool of answers, like "yes" and "when drinking" for the feature smokes.

Selecting a Feature will show the words which were most frequently used by the users to describe themselves.

Additionally users can write in their own words about themselves. They are given certain categories—the essays—to express a more personal side.

Selecting Essay will filter the essays that are taken into account. For example, selecting "asian" for the feature "ethnicity" will only count the words of users who have selected asian as part of their ethnicity.

Clicking the snow flake ❄ in the Features column will freeze the current feature filter. You can change the features and look at the new results but you will still be able to compare the new results with the frozen ones.

Tip: You can select multiple features by pressing Ctrl / Cmd while clicking when it makes sense (people can stem from multiple ethnicities but cannot be a smoker and a non-smoker at the same time.)

You can hover and hold over most things to get an explanation.
